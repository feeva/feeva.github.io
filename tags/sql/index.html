    лҚ°мқҙн„°лІ мқҙмҠӨ SQL нҠ№м„ұ л№„көҗ

    (2015л…„ 10мӣ” 1мқј нҳ„нҷ©: кө¬кёҖ л¬ём„ңлҘј мӣҗліёмңјлЎң н•ҳм—¬ нҢҢмқј лӮҙліҙлӮҙкё°лҘј нҶөн•ҙ нҺҳмқҙм§ҖлҘј м ңкіөн•ҳкі  мһҲм—ҲлҠ”лҚ° нҳ„мһ¬лҠ” нҢҢмқј нҳ•мӢқмқҙ л°”лҖҗ л“ұмқҳ л¬ём ңлЎң нҺҳмқҙм§Җк°Җ м•ҲлӮҳмҳӨкі  мһҲлӢӨ. мӢңк°„ лӮҳл©ҙ лӢӨмӢң мҲҳм •н•ҙм•јн•  л“Ҝ...)к·ё лҸҷм•Ҳ н”„лЎңм қнҠёлҘј л§Һмқҙ н•ҳлӢӨліҙлӢҲ лҚ°мқҙн„°лІ мқҙмҠӨлҸ„ м—¬лҹ¬к°Җм§ҖлҘј мӮ¬мҡ©н•ҙлҙӨлҠ”лҚ° м—¬лҹ¬ лӢӨлҘё к°ңл°ң л¶„м•јмІҳлҹј м ңн’Ҳк°„ м°Ёмқҙк°Җ кҪӨ лӮҳлҠ” л¶„м•јк°Җ л°”лЎң мқҙ мҳҒм—ӯмқҙлӢӨ. SQL кө¬л¬ё л“ұм—җ мһҲм–ҙм„ң нҒ¬кі  мһ‘мқҖ м—¬лҹ¬ м°Ёмқҙк°Җ мӮ¬лһҢмқ„ нҳјлһҖмҠӨлҹҪкІҢ н•ҳлӢӨліҙлӢҲ 2009л…„м—җ лҚ°мқҙн„°лІ мқҙмҠӨ SQL нҠ№м„ұ л№„көҗ л¬ём„ңлҘј л§Ңл“  м Ғмқҙ мһҲлӢӨ.

    mysqldump SQL нҳёнҷҳм„ұ лҶ’мқҙкё°

    mysqldumpлҘј мӮ¬мҡ©н•ҙ SQL мҠӨнҒ¬лҰҪнҠёлҘј л§Ңл“Өм–ҙлӮј л•Ң м—¬лҹ¬к°Җм§Җ лӘ©м Ғмқҙ мһҲкІ м§Җл§Ң лӢӨлҘё лҚ°мқҙн„°лІ мқҙмҠӨмҷҖмқҳ нҳёнҷҳм„ұмқ„ м—јл‘җм—җ л‘җлҠ” кІҪмҡ°к°Җ мў…мў… мһҲлӢӨ. мҳҲлҘј л“Өм–ҙ MySQLмқ„ к°ңл°ңмҡ©мңјлЎңл§Ң мӮ¬мҡ©н•ҳкі  лӢӨлҘё лҚ°мқҙн„°лІ мқҙмҠӨлҘј мӢӨмҡҙмҳҒмҡ©мңјлЎң мӮ¬мҡ©н•ҳлҠ” кІҪмҡ°к°Җ мһҲмқ„ мҲҳ мһҲлӢӨ.лҚ°мқҙн„°лІ мқҙмҠӨк°„мқҳ нҳёнҷҳм„ұ л¬ём ңлҠ” лӢЁмҲңн•ң л¬ём ңк°Җ м•„лӢҲлҜҖлЎң мқҙ кёҖм—җм„ңлҠ” мқјлӢЁ mysqldump лӘ…л №л§Ңмқ„ мЈјм ңлЎң SQL нҳёнҷҳм„ұмқ„ мөңлҢҖн•ң лҶ’мқҙлҠ” л°©лІ•мқ„ м•Ңм•„ліҙмһҗ. mysqldumpлЎң кІ°кіјл¬јмқ„ м–ҙлҠҗ м •лҸ„ нҳёнҷҳм„ұ мһҲкІҢл§Ң л§Ңл“Өм–ҙлҸ„ л§ҺмқҖ мҲҳкі лҘј мӨ„мқј мҲҳ мһҲлӢӨ. мҡ°м„  mysqldumpлҠ” лӢӨмқҢкіј к°ҷмқҖ нҳ•мӢқмңјлЎң лӘ…л №мқ„ мӢӨн–үн•ңлӢӨ.```sqldump [мҳөм…ҳ] [DBлӘ… [н…Ңмқҙлё”лӘ… ...]]```

    DB м§Ҳмқҳ кІ°кіјлҘј к°ҖлЎңлЎң нҺјм№ҳкё°

    мҳҲлҘј л“Өл©ҙ л¶Җм„ң н…Ңмқҙлё”кіј м§Ғмӣҗ н…Ңмқҙлё”мқҙ мһҲмқ„ л•Ң лӢӨмқҢкіј к°ҷмқҙ л¶Җм„ңлі„ м§Ғмӣҗ лӘ©лЎқмқ„ к°ҖлЎңлЎң нҺјміҗм„ң кө¬н•ҳкі  мӢ¶мқҖ кІҪмҡ° м–ҙл–»кІҢ н•ҙм•јн• к№Ң? л¶Җм„ң м§Ғмӣҗ мҙқл¬ҙл¶Җ к№Җл°•мӮ¬, мқҙмӮ¬мһҘ, мөңкі м§‘ мҳҒм—…л¶Җ мқҙлӢ¬мҲҳ, л¬ҙлҢҖлҰ¬ мһҗмһ¬л¶Җ к°•к°җм°¬, мқҙмҲңмӢ , мқ„м§Җл¬ёлҚ•, к№Җмң мӢ  кё°нҡҢк°Җ лҗҳл©ҙ лӢӨмқҢм—җ н’Җм–ҙм„ң м„ӨлӘ…н•ҙм•јкІ м§Җл§Ң мқјлӢЁмқҖ лӢӨмқҢ л§ҒнҒ¬лҘј м°ёкі н•ҳлҸ„лЎқ н•ңлӢӨ.
