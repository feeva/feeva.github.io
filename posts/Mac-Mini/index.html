전체 게시물

Mac Mini를 사다

불편한 점

  • 해상도 - 기본 글꼴 크기 등
  • 파일 작업 - 파일/폴더 찾기, 파일 정보 보기 등
  • 스크롤이 부드럽지 않아 moss라는 프로그램 별도 설치
  • 키보드와 마우스를 조화롭게 사용할 수 없다 - 대화창에서 esc, space, enter
  • 다른 프로그램간 전환은 cmd-tab키로 할 수 있으나 한 프로그램간 창 전환은
    • cmd-~로 하면 한번에 한창씩 전환되고
    • ctrl-down으로 하면 여러 창이 작게 보기로 나와서 어떤 창이 어떤 창인지 잘 안보임
  • 메뉴가 창에 안 붙어 있고 화면 상단에 있어서 불편

좋은 점

  • 잠금 화면, 화면 보호기에서 동영상을 재생할 수 있다. 삭제는 안되는 문제가 있다.

자잘한 버그들

  • VS Code에서 한글 입력할 때 받침이 밀리거나 사라지는 문제
  • 창 경계에서 마우스 커서가 크기 조정 커서 형태로 바뀌지 않는 문제
  • Night Shift가 일몰 시점에 켜지지 않는다. 윈도도 그랬다.

반면 윈도에서는

알림 메시지가 작업 표시줄에서 깜박 거리는 문제로 작업 표시줄을 자동 숨김으로 하는 의미가 없었으나 애플 도커는 잘 숨겨져 있음
